  antigravity
    add-context [ui] — Click the Add context button in the composer (opens file/URL picker for context attachment).
    cookies [ui] — List cookies on the Antigravity renderer (JS-visible via document.cookie).
    copy-code [ui] — Return the text of a code block in the current conversation. Default: last code block; pass --index N (1-based from top) to pick a specific one.
    copy-message [ui] — Return the text of the last assistant message (best-effort: walks up from the last visible Copy button).
    delete [ui] — Delete an Antigravity conversation by ID. Antigravity asks for confirmation; we click through it. Require --yes to actually delete.
    display-options [ui] — Open the Display Options menu and list its items.
    dump [ui] — Dump the DOM to help AI understand the UI
    extract-code [ui] — Extract multi-line code blocks from the current Antigravity conversation
    history [ui] — List visible Antigravity conversations from the sidebar
    idb-list [ui] — List IndexedDB databases on the Antigravity renderer.
    mark-read [ui] — Mark an unread Antigravity conversation as read. Fails if the row is already read or the postcondition cannot be verified.
    model [ui] — Read or switch the active model in Antigravity. Without arguments, reports the current model. With <name> (substring, case-insensitive), switches.
    nav [ui] — Click Go Back or Go Forward (Antigravity in-app history).
    new [ui] — Start a new conversation / clear context in Antigravity
    react [ui] — Click "Good response" or "Bad response" on the LAST assistant message.
    read [ui] — Read the latest chat messages from Antigravity AI
    recent-paths [local] — Show Antigravity's recently-opened folders/files (history.recentlyOpenedPathsList).
    rename [ui] — Rename an Antigravity conversation by ID (NOT YET IMPLEMENTED — see source comment).
    revert [ui] — Click the revert button (per-message revert for agent changes). Requires --yes (this modifies your workspace).
    send [ui] — Send a message to Antigravity AI via the internal Lexical editor
    settings [ui] — Click the Antigravity settings button (matched by data-testid="settings-button").
    settings-read [local] — Read Antigravity's user settings.json (theme, proxy, agCockpit, tfa.system.autoAccept, etc.).
    sidebar-toggle [ui] — Click Toggle Sidebar (collapses/expands the Antigravity sidebar).
    state-get [local] — Read one value from Antigravity's state.vscdb. Pass --workspace <id> for per-workspace.
    state-keys [local] — List keys in Antigravity's globalStorage state.vscdb (VSCode-style). Pass --workspace <id> to query a per-workspace DB. Works while Antigravity is closed.
    status [ui] — Check Antigravity CDP connection and get current page state
    storage-get [ui] — Read a single localStorage / sessionStorage value on the Antigravity renderer.
    storage-keys [ui] — List localStorage / sessionStorage keys on the Antigravity renderer (CDP).
    toggle-aux [ui] — Toggle the Auxiliary Pane (Antigravity's secondary panel for code/preview).
    watch [ui] — Stream new chat messages from Antigravity in real-time
    workspaces-list [local] — List Antigravity workspaceStorage entries (each represents a previously-opened folder).

    task-fs-show [local] — Show the workspace tree at a given chat-turn ref (via git ls-tree). Pass --turn <turn-id> to pick a turn; otherwise the latest after-chat-turn ref.
    task-fs-turns [local] — Show the chat-turn timeline for a Trae SOLO task as git tags (before-chat-turn-* / after-chat-turn-*).
    user-rules [local] — Print Trae SOLO user rules (~/.trae/user_rules.md).
    workspaces-list [local] — List Trae SOLO workspaceStorage entries (~/Library/.../TRAE SOLO/User/workspaceStorage/<uuid>/), resolving each workspace.json to its single-folder path or multi-folder workspace target. Works while Trae is closed.
